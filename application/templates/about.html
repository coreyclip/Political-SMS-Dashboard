About

This projects originated from around 2020 when I started getting text messages from the Trump campaign. Some of the texts were frankly pretty wild, and it made me realize that while a lot of political campaigning in the form of ads or even brochures are publicly available, sms campaigning wasn't. In the end these are targeted pieces of outreach from political campaigns towards their more involved supporters. In the interest of public awareness and research, I decided to set up this website to be a public facing portal to the text messages I've been receiving. I have things automated to where the website should have incoming text messages being refreshed on a daily basis. Be sure to check back regularly if this project interests you as I will be adding new features as I have time.

How you can help

I try to add new campaigns to be monitored when I find them. Thanks in advance :) also if you're learning web development or are otherwise technical feel free to reach out if you'd like to help out with dev work!

Understanding the sentiment scores: Positivity, Negativity, Intensity

These are computed using the Valence Aware Dictionary and sEntiment Reasoner (VADER) which is used for analyzing sentiments expressed through social media, it is a lexicon and rule-based sentiment analytics tool. You can learn more about it here.

  • Positivity relates to weather the content of the text is more frequently expressing optimism or happiness
  • Negativity relates to weather the content more frequently expresses pessism or unhappiness
  • Intensity is what this website calls what VADER refers to as compound, which is a measure of intensity. A very positive or negative text can vary in terms of it's compound, as in how intensley it expresses anger or optomism
